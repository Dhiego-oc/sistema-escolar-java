## 📚 Sistema Escolar em Java

Projeto desenvolvido com o objetivo de praticar lógica de programação e manipulação de arquivos em Java.

## 🚀 Funcionalidades

- Cadastro de notas de um aluno
- Validação de notas (entre 0 e 10)
- Cálculo da média final
- Classificação do aluno:
  - Aprovado
  - Recuperação
  - Reprovado
- Armazenamento dos dados em arquivo texto
- Leitura e exibição dos dados salvos

## 🛠️ Tecnologias utilizadas

- Java
- Scanner (entrada de dados)
- FileWriter / PrintWriter (persistência em arquivo)
- Tratamento de exceções

## 📚 O que eu aprendi

- Estruturas de repetição ("for", "while")
- Estruturas condicionais ("if/else")
- Validação de entrada de dados
- Manipulação de arquivos em Java
- Organização básica de código

## ▶️ Como executar o projeto

1. Compile o código:
   
   javac SistemaEscolar.java

2. Execute:
   
   java SistemaEscolar
